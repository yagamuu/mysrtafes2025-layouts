# Mystery Dungeon RTA FES 6 Layouts
第6回不思議のダンジョンシリーズRTAフェスというイベント用に作成したNodeCGのbundleです。
This is the [NodeCG](http://github.com/nodecg/nodecg) bundle used for the streaming overlay in Mystery Dungeon RTA FES 6.

## Usage
```
npm install
docker compose up
```
